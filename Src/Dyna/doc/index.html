2. Release notes

Release 1.1.1, October 10, 2007
- Fixed bug causing object attributes to not be summed and stored in the object on the left-hand-side.
- Added sqrt function to TBL.
- Improved the scoping and garbage collection. Variables that are out of scope are now automatically removed from the stack.

Release 1.1
- Added export to OWL (OWL DL).

Release 1.0
- Initial release.



3. Installation

Unzip the dyna.zip file into the <Protege_Dir>/plugins directory

4. Uninstallation

Delete the <Protege_dir>/plugins/ca.uregina.engg.eil.dyna directory

5. Running Dyna
  1. Run Protege
  2. Click Project->Configure
  3. Check Dyna
  4. Click OK
